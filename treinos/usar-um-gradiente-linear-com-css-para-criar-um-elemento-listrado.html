Para este treino, usamos a função repeating-linear-gradient(), que é muito semelhante a linear-gradiente, com a principal diferença de que repete o padrão de gradiente especificado.

O valor do ângulo é a direção do gradiente. As interrupções de cor são como valores de largura que marcam onde ocorre uma transição e são fornecidas com uma porcentagem ou um número de pixels.

Já para este conteúdo, foram definidos os distanciamentos para ambas as margens laterais.
